Frequently asked questions

  • How do I get started?
    A: Type your web search terms into the text box on top of this page and click the button "Start 2charts.com session". When prompted, log on with your Google account. If you find a table on a web page, click on it and see what happens...
  • Can I use 2charts.com with any web page on the internet?
    A: YES*.
    From the 2charts.com home page, just navigate to the web page of your choice. Then click on the data table that you want to visualize - an appropriate chart will appear as a modal popup.
  • All this works without any installation procedure?
    A: YES*.
    Your web browser already knows what to do. Just sit back, surf the web and click on a table if you want to create a chart.
* The rendering engine used by 2charts.com leverages intelligent algorithms to interpret tabular data on the internet and derive appropriate charts from that data. Although this engine makes its best effort to correctly "parse" the table data, there is no guarantee that it will get the results that you expect. The challenge is the vast variety of more or less exotic tables out on the web. We will constantly improve those algorithms in order to increase 2charts.com's successful rendering ratio.
